Overview
Welcome to the Movie Web App! This web application allows users to explore a collection of movies, view details about each movie, and search for their favorite films. It provides a user-friendly interface to enhance the movie-watching experience.

Features
Browse Movies: Users can browse a list of available movies.
Movie Details: View detailed information about each movie, including the title, release date, genre, and a brief description.
Search Functionality: Users can search for specific movies using the search bar.
Responsive Design: The application is designed to work seamlessly on desktops, tablets, and mobile devices.
